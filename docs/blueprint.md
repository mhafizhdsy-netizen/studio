# **App Name**: GenHPP

## Core Features:

- User Authentication: Secure user authentication using Firebase (email/password and Google login) with isolated user data in Firestore.
- Dashboard History: Interactive dashboard displaying calculation history stored in Firestore, with options to view details, delete, and edit calculations.
- HPP Calculator: Interactive calculator for computing the cost of goods sold (HPP) based on entered parameters. Includes a breakdown in pie chart form, and automated currency formatting.
- Social Sharing: Ability to share calculation results to social media, with options to generate a shareable link or export to PDF/XLSX.
- Community Sharing: Option for users to share calculation results (without personal data) publicly to inspire other young entrepreneurs; use Firestore subcollection /publicCalculations.

## Style Guidelines:

- Primary color: Electric blue (#2D9CDB) for a modern and energetic feel.
- Secondary color: Jet black (#1C1C1C) for contrast and a sleek design.
- Accent color: Neon green (#00FF88) for highlighting important elements and CTAs.
- Font: 'Poppins', a geometric sans-serif font that gives a contemporary, precise, fashionable, avant-garde look; suitable for headlines and short amounts of text
- Card-based design with ample whitespace, ensuring a responsive and mobile-first approach.
- Subtle hover animations and a visual progress bar during calculations to enhance user engagement.
